도서 관리 시스템
1. 도서 정보를 입력 후 등록할 수 있다.(이미지, 도서명,저작자, 출판사, ISBN, 가격, 수량, 구매가능 연령) -- 엑셀 파일 업로드 가능, 이미지는 없을 시 기본 이미지
2. 도서의 목록을 보여줄 수 있다(10개씩)
3. 도서의 목록(도서명, 저작자, 출판사, ISBN, 가격, 수량, 구매가능 연령, 이미지) 행 클릭 시 해당 도서의 상세 정보를 보여줄 수 있고 그 레이어에서 수정 및 삭제 또한 가능하다.(도서 삭제 시 특정 플래그 값을 바꿈으로써 목록에서 제외한다)
4. 목록 및 상세 정보에서 해당 도서의 이미지를 클릭 시 도서를 큰 사이즈로 보여준다.
5. 도서 판매 시 수량을 기재할 수 있고, 최종 판매 금액을 입력 확인 후 판매 의사를 결정한다.
6. 도서 입고 클릭 시 해당 도서가 몇개 입고 되었는가를 입력할 수 있고, 기존 수량에서 해당 입력 수만큼 더한다.
7. 도서 판매 기록을 확인할 수 있고, 매출 또한 시각화하여 보여줄 수 있다.
