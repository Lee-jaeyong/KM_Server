* 로그인
- 로그인 시 아이디 비밀번호 입력 후 메인 대쉬보드에 접근 가능하다. => 로그인 후 사용자의 권한(학생, 교수)을 확인 후 권한에 따라 다른 UI를 보여준다.

* 강의
[
- 강의 등록 시 상단에 자신의 학과를 보여주고, (강의명, 시작일, 종료일, 강의 계획서 업로드, 강의 계획서 양식 다운로드, 
내용, 강의 종류<교양,전공>, 댓글 허용 여부, 메뉴 종류, 강의 사용 여부) 를 입력 받도록 한다.

- 강의 등록 후 등록한 강의의 내용 페이지(수업 신청 현황, 과제 현황, 등록한 수업 정보, 수정[수정 시 수정 페이지로 이동] 및 삭제 가능)로 사용자를 이동시킨다.
]

* 참고 자료
[
- 참고 자료 등록 시 참고 자료에 대한 수업을 보여주고 (제목, 내용, 개시 상태, 파일 및 이미지 업로드)를 할 수 있도록 한다.

- 등록 후 등록한 수업에 대한 참고자료를 보여주는 목록 페이지
(참고자료 코드, 제목, 내용 일부분, 등록일, 조회 수, 게시 상태, 버튼[참고자료 삭제, 수정])로 이동한다.
- 참고 자료 목록 해당 ROW 클릭 시 참고 자료 보기 페이지로 이동한다. 페이지에는 수정 및 삭제 버튼이 존재한다.
]

* 공지 사항
[
- 공지사항 등록 시 공지사항에 대한 수업을 보여주고 (제목, 내용, 개시일, 파일 및 이미지 업로드)를 할 수 있도록 한다.

- 등록 후 등록한 수업에 대한 공지사항을 보여주는 목록 페이지
(공지사항 코드, 제목, 내용, 등록일, 조회 수, 개시 상태, 저장 및 임시저장 상태, 버튼(공지사항 수정, 삭제)
- 공지사항 목록 해당 ROW 클릭 시 공지사항 보기 페이지로 이동한다. 페이지에는 수정 및 삭제 버튼이 존재한다.
]

* 과제
[
- 과제 등록 시 상단에 과제에 대한 수업을 보여주고, (과제명, 과제 시작일, 과제 마감일, 내용, 마감 이후 제출 가능 여부,업로드 파일 확장자, 학생 제출 과제 공개 상태)
를 입력 받도록 한다.

- 과제 등록 후 과제 목록 페이지(과제 번호, 과제명, 과제 시작일<이미 시작했는가에 대한 상태>, 마감일<몇일 남았는지에 대한 상태> --- 마우스 오버시 마감 후 제출 가능 여부 출력
, 조회 수, 과제 생성일, 파일 및 이미지 여부, 버튼[과제 삭제, 과제 수정], 제출 현황 보기<현재 몇명 제출했는가 표시>) 로 이동한다.

- 과제 목록에 과제 해당 ROW 클릭 시 과제 보기 페이지로 이동한다. 과제 보기 페이지에는 수정 버튼 및 삭제 버튼이 존재한다.
- 과제 삭제 시 바로 삭제하지 않고 7일동안 보관 후 삭제 처리 하도록 한다.
]
